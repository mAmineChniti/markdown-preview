# Markdown Preview

Markdown Preview is a web application that allows users to preview their Markdown files side by side. This tool is ideal for writers, developers, and anyone who frequently works with Markdown. With real-time rendering, you can see how your Markdown content will look once itâ€™s converted to HTML.

## Features

- **Real-time Preview**: As you type your Markdown content on one side, the preview is updated instantly on the other side.
- **Syntax Highlighting**: Supports syntax highlighting for code blocks.
- **Live URL Links**: Clickable links in the preview.
- **Image Rendering**: Displays images embedded in your Markdown.
- **Responsive Design**: Works on both desktop and mobile devices.
